Simple report, list of courses, choose a course:

• registered users
• alert if not connected in the last 15 days
• Number of quizzes completed (completion) / total quizzes in the course
• Number of scorms completed (completion) / total scorms in the course
• Number of resources completed (completion) / total resources in the course

You can click on quiz or scorms, which shows another report with specific information on each of the quizzes or each of the scorms.
